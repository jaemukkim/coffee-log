# coffee-log

## 🚀 주요 기능 (Key Features)

### 1️⃣ 스마트 퀵 기록 및 재고 관리
* **원클릭 기록:** Streamlit 버튼 하나로 '오늘의 커피'를 즉시 기록합니다.
* **자동 재고 차감:** 기록 시 `MySQL`의 `stock_count`가 자동으로 `-1` 차감되는 `UPDATE` 로직을 구현
* **재고 부족 알림:** 남은 캡슐이 5개 이하일 경우, 대시보드 상단에 빨간색 경고 메시지를 띄워 재구매 시점을 놓치지 않도록 돕습니다.

### 2️⃣ 데이터 기반 취향 분석 리포트 (Plotly 활용)
* **맛 분포 분석:** 한 달간 소비한 캡슐의 산미와 강도 데이터를 집계하여 레이더 차트(Radar Chart) 또는 바 차트로 시각화합니다.
* **주간 카페인 추이:** 요일별 커피 소비량을 시각화하여 본인의 카페인 섭취 패턴을 분석합니다.

### 3️⃣ 지능형 캡슐 추천 엔진
* **소비 패턴 분석:** 최근 일주일간의 소비 데이터를 분석하여 사용자 취향에 맞는 캡슐을 제안합니다.
* **조건부 로직:** (예) "최근 강도가 높은 캡슐 위주로 드셨네요. 오늘은 산미가 있는 가벼운 캡슐은 어떠신가요?"와 같은 맞춤형 문구를 제공합니다.
